# Cursor Prompts

This file stores reusable prompts for JamiiFlow development phases.

## Phase Control Prompt
"Work in strict phased order. Implement only the requested phase, keep the app compiling, and update docs after each major step."

## Code Quality Prompt
"Use TypeScript with explicit types, validate input with Zod, and keep modules focused and testable."

## Documentation Prompt
"After implementation, update architecture, setup, API, and database docs to match the current code exactly."
